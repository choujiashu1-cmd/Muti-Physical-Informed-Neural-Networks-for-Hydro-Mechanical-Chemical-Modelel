# 一维 HMC 耦合模型代码与数据说明

## 项目简介

本文件夹用于存放一维 HMC（Hydro-Mechanical-Chemical）耦合模型的小论文代码、计算结果和过程记录。当前内容以一维正演为主，已经整理出均质算例、解析解对比、单参数非均质算例和三参数非均质算例，并保留了后续扩展到反演与二维模型的目录框架。

本项目重点关注一维深度-时间域中的三类响应：

- 孔压 `U`
- 浓度 `C`
- 表面沉降 `Settlement`

从现有脚本来看，核心研究路线是：先建立均质一维 HMC 正演模型，再利用 MPINN 与 RAR（Residual-based Adaptive Refinement）提高训练质量，随后推广到参数随深度线性变化的非均质情形，并为后续反演准备虚拟观测数据。

## 主要文件说明

### 1. 均质计算及验证

#### `一维正演/均质计算及验证/HMCForward.py`

这是当前最完整的均质一维 HMC 正演脚本，主要功能包括：

- 建立一维 HMC 耦合控制方程
- 使用 DeepXDE + PyTorch 训练 MPINN
- 引入 `log(1 + 100t)` 的时间特征变换，加强早期时段分辨率
- 采用 RAR 自适应重采样机制，提高高残差区域学习效果
- 预测 1、10、20、50 年时的孔压与浓度剖面
- 计算 0.01 年到 100 年的表面沉降时程
- 导出均质算例的结果数据

该脚本导出的主要结果文件为：

- `MPINN_Profile_Data_1_10_20_50_RAR.csv`
- `MPINN_Settlement_Data_RAR.csv`

#### `一维正演/均质计算及验证/HMCForward_AnaCompare.py`

该脚本用于将 PINN 结果与解析解进行对比验证，主要功能包括：

- 定义均质一维 HMC 问题的参数类和系数类
- 构造解析解求解器
- 读取 `HMCForward.py` 导出的 CSV 结果
- 绘制浓度剖面、孔压剖面和表面沉降时程的对比图

这个脚本说明目前的均质算例不仅有数值结果，也包含了与解析解的验证环节，适合作为论文中“模型正确性验证”的基础材料。

### 2. 非均质计算

非均质部分主要位于 `一维正演/非均质计算/`，可以分为两类。

#### 单参数非均质

包含三个独立算例：

- `压缩模量变化`
- `扩散系数变化`
- `渗透系数变化`

这三个算例的共同特点是：

- 只让一个关键参数随深度线性变化
- 其余参数保持均质
- 训练流程与均质算例基本一致，仍使用 MPINN + RAR
- 输出剖面 CSV，并打印后续反演可用的虚拟观测值

从脚本内容看，虚拟观测值通常取自：

- 观测深度 `z = 0.8 m`
- 观测时刻 `10, 20, 50` 年

这些结果明显是为“一维反演”做前期准备。

#### 三参数非均质

`三参数非均质正演.txt` 中同时考虑了以下参数的深度线性变化：

- 渗透系数 `kh`
- 压缩模量 `E`
- 扩散系数 `D0`

这是当前非均质部分里更综合的算例，适合对应论文中更复杂的功能梯度或分层近似场景。

## 模型与训练设置概览

根据现有均质主脚本，当前模型的关键设置如下：

- 空间区间：`z ∈ [0, 1]`
- 时间区间：`t ∈ [0, 1]`
- 时间缩放：`t = 1` 对应 `100` 年
- 网络结构：`[3] + [80] * 8 + [2]`
- 激活函数：`tanh`
- 后端：`PyTorch`
- 精度设置：`float64`

均质主算例中使用的典型参数包括：

- 初始参考孔压 `p0 = 50000 Pa`
- 初始参考浓度 `c0 = 10`
- 渗透系数 `kh = 1.0e-10`
- 压缩系数 `mv = 5.0e-7`
- 化学相关系数 `mc = 1.05e-4`
- 分配系数 `Kd = 8.142e-4`
- 扩散系数 `D0 = 5.0e-10`
- 弹性模量 `E = 1.49e6`
- 泊松比 `nu = 0.3`

训练流程为三阶段：

1. Adam 预训练 `5000` 次
2. 进行 `5` 轮 RAR，每轮搜索高残差点并继续训练 `2000` 次
3. 使用 `L-BFGS` 做最终收敛优化，`maxiter = 10000`

## 运行环境建议

建议使用如下环境：

- Python 3.10 或更高版本
- `deepxde`
- `torch`
- `numpy`
- `scipy`
- `pandas`
- `matplotlib`

可参考安装方式：

```bash
pip install deepxde torch numpy scipy pandas matplotlib
```

脚本中已显式设置：

```python
os.environ["DDE_BACKEND"] = "pytorch"
```

因此默认使用 PyTorch 作为 DeepXDE 后端。若本机有 GPU，可自动检测并加速训练；若没有 GPU，也可以在 CPU 上运行。

## 使用方法

### 1. 运行均质正演

在 `一维正演/均质计算及验证/` 目录下运行：

```bash
python HMCForward.py
```

运行后可得到：

- 浓度剖面图
- 孔压剖面图
- 表面沉降时程图
- `MPINN_Profile_Data_1_10_20_50_RAR.csv`
- `MPINN_Settlement_Data_RAR.csv`

### 2. 进行解析解对比

在生成均质算例结果文件后，运行：

```bash
python HMCForward_AnaCompare.py
```

该脚本会读取前一步导出的 CSV，并绘制 PINN 与解析解的对比图。

### 3. 运行非均质算例

非均质目录中的代码目前保存在 `.txt` 文件中，内容形式更接近 Notebook/Colab 记录，文件前部保留了 `! pip install ...` 语句。因此：

- 如果在 Jupyter 或 Colab 中使用，可以直接按代码块运行
- 如果在本地 Python 环境中运行，建议先另存为 `.py` 文件
- 本地运行前，建议删除或注释掉 `! pip install ...` 这类 Notebook 专用语句

非均质算例运行后通常会输出：

- 不同年份的孔压与浓度剖面
- 对应的沉降时程
- `Forward_Linear_FGM_Profiles*.csv`
- 后续反演可用的虚拟观测值

## 输出文件说明

### 均质算例输出

#### `MPINN_Profile_Data_1_10_20_50_RAR.csv`

该文件记录深度方向剖面数据，字段包括：

- `Depth_m`
- `PorePressure_kPa_1yr / 10yr / 20yr / 50yr`
- `Concentration_kg_m3_1yr / 10yr / 20yr / 50yr`

用于展示不同时间下的孔压与浓度分布。

#### `MPINN_Settlement_Data_RAR.csv`

该文件记录沉降时程数据，字段包括：

- `Time_years`
- `Settlement_mm`

用于绘制表面沉降随时间变化的曲线。

### 非均质算例输出

#### `Forward_Linear_FGM_Profiles.csv` / `Forward_Linear_FGM_Profiles (1).csv`

该类文件记录非均质算例在多个时刻的剖面结果，字段通常包括：

- `Depth_m`
- `PorePressure_kPa_1yr / 10yr / 20yr / 50yr`
- `Concentration_C_C0_1yr / 10yr / 20yr / 50yr`

其中浓度一般使用归一化形式 `C / C0` 表示。
